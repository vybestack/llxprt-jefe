  no-match states.

  section order Description/body → Reviews → Checks → Comments(+loading) → New

  `PrFilterControlsProps` (6 filter fields band, hidden when `!visible`).

  `PR_DETAIL_HEADER_ROWS == DETAIL_HEADER_ROWS`, `prs_pane_rows` delegating to
